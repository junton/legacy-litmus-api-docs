Overview

The Email Analytics SOAP API is a special API that allows partners to resell Litmus Email Analytics via their own interface. Using this Web Service you can create an unlimited number of tracking codes, and pull back all of the detailed tracking data we collect.

 

The bulk of the documentation resides within our Web Service. From here you can find the address of the WSDL and detailed information about each method, expected input and the objects returned.

 

The Process

First, call Create to create a new Email Analytics report with a unique HTML snippet for your campaign. Now read the contents of the BugHtml field within the returned object. This HTML should be included within the email campaign that's about to mailed out, directly above the closing body tag "</body>". Be sure to include it at the end of the email, but within the body.

 

Once your email has been sent out, you'll start to see data returning within minutes. A list of reports can be found here. You can format this data however you wish. By way of example, see this sample report that shows how we present this data within the Litmus interface.
