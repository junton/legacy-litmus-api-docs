Our pageTest object is now as customised and prepared as it can be. No other properties can be set (that'll have an effect, anyway).

 

Ordering a new page test is very straightforward. We've created a CreatePageTest method for you to send your pageTest object to us:

 

pageTest = LitmusApi.CreatePageTest("api-key", "api-pass", pageTest)

 

There are a few things happening at once here, so here's a breakdown of each.

 

  1. First we call the CreatePageTest method of the Litmus Web Service and pass in our API key and API pass.
  2. We then pass, as the thrid parameter, the pageTest object we created above.
  3. Finally, the CreatePageTest method actually returns an object which is an instance of the PageTest class, so we're assigning that return value to our existing emailTest.
