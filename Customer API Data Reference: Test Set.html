Data Reference: Test Set

 

Parameter:     created_at

Type:              date/time

RO/RW:          read-only

The XML formatted time at which this test was created in Litmus

 

 

Parameter:     updated_at

Type:              date/time

RO/RW:          read-only

The XML formatted time at which this test was last updated in Litmus

 

 

Parameter:     id

Type:              integer

RO/RW:          read-only

The identifier assigned to this test.

 

 

Parameter:     name

Type:              string

RO/RW:          read-write

The name that appears next to this test in the Litmus web interface. By default, for page tests this is set to the page title, for email tests it is set to the email subject line. This can be changed by updating a test once it has been created.

 

 

Parameter:     service

Type:              string

RO/RW:          read-only

The testing service used to generate this test. At the moment this will only return page or email as these are the only services currently offered.

 

 

Parameter:     state

Type:              string

RO/RW:          read-only

The overall state of this test. This is here for convenience and will indicate the state of the most recent test set version. If you create a new version of an existing test then this will change back to the processing state. Valid values for this parameter are: complete, processing and for email tests that have not been received by Litmus yet: waiting

 

 

Parameter:     public_sharing

Type:              boolean

RO/RW:          read-write 

Indicates whether this test is shared publicly. When set to true then a public_path parameter will also be set and indicate the url needed to access the tests via the public interface. This parameter can be set by updating a test once it has been created. If you want to generate a new public_path (perhaps the old one was accidentally leaked) you can switch public_sharing off and on again and it will generate a new url.

 

 

Parameter:     public_path

Type:              string

RO/RW:          read-only

The full url to access this test set publicly hosted on the Litmus web interface. Will use https if the account has SSL and it is enabled.

 

 

Parameter:     url_or_guid

Type:              string

RO/RW:          read-only

The url of the page being tested if this is a page test, for email tests it is the email address to send the test. This is specified as a convenience accessor to the most recent test version. Normally the page url won't change between versions, but each new email test generates a new guid.
