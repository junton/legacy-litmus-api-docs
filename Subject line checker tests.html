Introduction

The subject line checker is very simple to use within your own application. First add a reference to our javascript, then simply call the method generateSubjectLineUrl for each type of result you want.

 

Adding the Litmus javascript

First, add the following to the HEAD of each page you'd like to use the subject line checker in:

 

<script src="http://media.emailtests.com/litmus-subject.js"></script> 

 

If you prefer to use SSL:

 

<script src="https://s3.amazonaws.com/media.emailtests.com/litmus-subject.js"></script>

 

Using the Litmus javascript

Next, simple call the javascript function generateSubjectLineUrl:

 

function generateSubjectLineUrl(client, previewType, subject, plainBody, htmlBody, fromName)

 

Parameter name  Description  Example value 
client  One of a set number of clients available to test on. See the clients and types table below.  "ol2003" 
previewType  Either "subject" or "toast". "subject" is the subject line preview, "toast" is the little preview window that appears when a new email arrives in some email clients.  "subject" 
subject  The subject of the email to be tested.  "October 2011 Newsletter" 
plainBody  The plain text version of the email's body.

"======

Welcome to ..." 
htmlBody  The HTML version of the email's body.  "<h1>Welcome to...</h1>" 
fromName  The sender's name, not email address. For example: "Paul Farnell", not "paul@litmus.com" or "Paul Farnell <paul@litmus.com>".  "Paul Farnell" 

 

The following clients and types are available to use. This list may change/expand in the future.

 

Description  client previewTypes available
Outlook 2003 ol2003  toast, subject 
Outlook 2007  ol2007  toast, subject 
Outlook 2010  ol2010  toast, subject 
Hotmail  hotmail  subject 
Gmail  gmail  subject 
Yahoo  yahoo  subject 
BlackBerry  blackberry  subject 

 

To use the function, create an IMG element in HTML and then assign its src attribute the result of the call to generateSubjectLineUrl:

 

document.getElementById('preview').src = generateSubjectLineUrl('ol2003', 'toast', 'This is the subject', 'This is the plain body', '<h1>This is the html body</h1>', 'Matthew Brindley');

 

This assumes we have an IMG with an id of "preview".

 

Sample code

Below is a sample HTML page that will, on body load, set the src of its only image element to the generated url.

 

<html>
  <head>
    <script src="http://media.emailtests.com/litmus-subject.js"></script> 
     <script type="text/javascript">
        function bodyOnLoad(){
          document.getElementById('preview').src = generateSubjectLineUrl('ol2003', 'toast', 'This is the subject', 'This is the plain body', '<h1>This is the html body</h1>', 'Matthew Brindley');
        }
     </script>
  </head>
  <body onLoad="bodyOnLoad();">
    <img id="preview"/>
  </body>
</html>
