Data Reference: 

 

Parameter:     inline_css

Type:              boolean

RO/RW:          read-only

When reading an email test, this indicates whether the test was run through our css pre-mailer, which compiles a head <style> block into inline css attributes.

 

 

Parameter:     version

Type:             integer

RO/RW:          read-only

This is the version number of the test version, numerous test versions will exist in sequence within one test set. You can use this attribute to chronologically order versions of a test when displaying to the end user.

 

 

Parameter:     received

Type:              boolean

RO/RW:          read-only 

For an email test, this attribute indicates whether the email you sent has been received by Litmus. Once it has been received our system will set this to true and testing will begin.

 

Parameter:     url_or_guid

Type:              string

RO/RW:          read-only

The url of the page being tested if this is a page test, for email tests it is the email address to send the test.
