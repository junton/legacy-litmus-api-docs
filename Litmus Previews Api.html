A friendly reminder: The Previews Api is only for use with a reseller account. It does not work with a regular Litmus account. if you're interested in offering Litmus to your own users, we'd love to hear from you! 

 

Technical Documentation

 

The Previews Api is a REST based web service.  You'll make POST and GET requests to interact with the service.  

 

The endpoint and documentation are available at https://previews-api.litmus.com

 

Authentication

 

All requests will be authenticated using HTTP Basic Authentication using your Api Key and Api Password.

 

To test your credentials the following curl request should always return some amount of clients:

 

curl -u ApiKey:ApiPassword https://previews-api.litmus.com/api/v1/EmailTests/TestingApplications
