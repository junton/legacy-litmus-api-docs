The Setup

Before you can begin recording hits, you'll need to create a campaign using the API. Below we've explained the most common way of creating campaigns in EA, and fetching aggregate data.

 

$ var campaign = api.CreateIndividual(user, pass)

 

CreateIndividual

This method creates a new Email Analytics campaign with individual-level tracking, it returns a Campaign object. The returned Campaign object contains a BugHtml property that you'll need to modify before embedding. 

 

$ campaign.BugHtml

<img src="https://rudi5xvg.emltrk.com/rudi5xvg?d=[UNIQUE]" width="1" height="1" border="0" />

 

 

Within the BugHtml, find every instance of [Unique] and replace it with your email address merge tag. For example, it might be %%email%% or (email_address). Whatever your merge tag is, make sure that the [Unique] placeholder has been completely removed, including the square brackets.

 

$ var campaign.BugHtml = replace(campaign.BugHtml, '[Unique]', '%%email%%')

 

Once your BugHtml is ready, you'll need to insert it into your HTML email before it's sent. Ideally, it should be place directly above the close body tag.
