Introduction

 

 For Timeline the work flow is:

 

  • Create an EmailTest with Litmus
  • Request the Timeline data

 

Details 

 

Step 1 :: Start a Litmus email test

 

See REST Email Previews for details.

 

Step 2 :: Getting your timeline results

 

You can get the timeline results for your email test by making a GET request to https://previews-api.litmus.com/api/v1/EmailTests/{id}/Timeline

 

 

Understanding your Timeline Result

 

A timeline result contains three objects all with the same set of properties.  Each object simulates the performance you should expect on that platform based on response times from your servers.

 

Desktop/Mobile3G/MobileLTE Object

 

MaxConnections Constant number of concurrent connections for the platform
DataRate Constant estimated maximum rate of data transmission for the platform in bits per second
Started  The time the first timeline event started
Completed  The time the last timeline event completed
TotalDuration The duration between Started and Completed
Events

Event specific details 

 

Events Object

 

Resource The name of the resource (typically an image url) 
Size  The size of the resource in bytes
Started  The time the event started 
Completed  The time the event completed 
TimedOut  A boolean indicating the event timedout 
Error  A boolean indicating the event could not be completed due to error 
ErrorMessage  A description of the error
