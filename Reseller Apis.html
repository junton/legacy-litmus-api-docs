REST Previews API

Reference docs:  https://previews-api.litmus.com/

 

Launched in March of 2014 this is our newer REST based api for creating email previews, spam tests, web page previews, link tests and code analysis tests.

 

Click here for documentation for this api.

 

Email Analytics API

Reference docs:  https://analytics-api.litmus.com

 

Launched in January of 2014 this is our REST based api for creating Email Analytics tracking codes and getting reports on your email campaigns.

 

Detailed documentation coming soon...

SOAP Previews API 

(this api is currently being sunset, new implementations should use the REST Previews API described above)

Reference docs:  https://soapapi.litmusapp.com/2010-06-21/api.asmx

 

Launched in June of 2010 this is our SOAP based api for creating email previews, spam tests and web page previews.

 

Click here for documentation for this api.

 

Subject Line Checker Integration
