Customer REST API Method: versions

Returns all versions for a specified test.

 

URL:

https://#{company}.litmus.com/tests/#{test-id}/versions.xml

 

HTTP Method(s):

GET

 

Parameters:

  • test-id.  Required. The ID of the test you want to see versions for.
